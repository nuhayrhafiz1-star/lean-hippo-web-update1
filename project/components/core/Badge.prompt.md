Status / category badge — monospaced, tracked, uppercase, reads as a system label.

```jsx
<Badge tone="ok">System OK</Badge>
<Badge tone="pending">Awaiting approval</Badge>
<Badge tone="alert" variant="dot">Leak detected</Badge>
<Badge tone="cobalt" variant="outline">AI Layer</Badge>
```

Tones: `neutral | cobalt | ok | pending | alert`. Variants: `soft` (filled), `outline`, `dot`. Use for live status, counts, and node labels — not for long text.
