Text input on the inset surface with cobalt focus edge.

```jsx
<Input label="Organisation" placeholder="LeanHippo Pty Ltd" />
<Input label="Org ID" defaultValue="LH-0042" mono />
<Input label="Email" error="Enter a valid work email" />
```

Supports `iconLeft`, `hint`, `error`, `disabled`, and `mono` (for IDs / codes). Focus shows the controlled cobalt ring.
